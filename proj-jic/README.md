#   Solution Architecture
Please consider the below when adding new project under the main JIC solution:

*   All projects must be fully named
*	All projects me be prefixed by JIC.
*	Each Solution Folder MUST be mapped to a physical folder except for solution items. Whenever required, (ASK me to show you how).
*	All projects must have assembly meta-data that matches the project’s name.
*	Ensure NOT to place any static or data file in this solution. 
*	All utilities to be done as PowerShell.
*   All utilities will be setup in another solution (proj-jic-automation)
